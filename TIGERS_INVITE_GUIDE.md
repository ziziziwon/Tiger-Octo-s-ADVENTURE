# 🎟 타이거즈의 초대장 (Tigers' Secret Invite)

## 📖 개요
"타이거즈의 초대장"은 모든 추억을 수집한 플레이어만이 경험할 수 있는 특별한 **After Credit Scene**입니다.

단순한 엔딩이 아닌, **"관객이 세계의 일부가 되는 순간"**을 연출하는 포트폴리오 안의 또 하나의 공연입니다.

---

## 🌟 체험 플로우

### 1️⃣ 트리거 조건
- **모든 아이템 수집** (10개+)
- **모든 메모리 카드 수집** (6개+)
- 조건 충족 시 **갤러리 페이지**에 반짝이는 초대장 카드 자동 등장

### 2️⃣ 초대장 카드 (InviteCard)
- 💎 **디자인**: 황금빛 그라데이션, 반짝이는 애니메이션, 홀로그램 효과
- 🎨 **인터랙션**: 
  - 마우스 호버 시 살짝 회전 + 그림자 확대
  - 클릭 시 **Glitch 전환 효과** (RGB shift + 화면 깜빡임)
- 🔗 **동작**: `/invite` 페이지로 이동

### 3️⃣ 초대장 페이지 (InviteScene)
- 🌌 **배경**: Canvas 기반 별 파티클 시스템 (150개의 별, 반짝임 애니메이션)
- 🌕 **연출**: 
  - 픽셀 밤하늘 (#0b0c10)
  - 파스텔 색상의 별들 (핑크, 블루, 옐로우, 로즈, 민트)
  - 달 장식 (opacity 펄스 애니메이션)
- 📜 **메시지**: 
  > "극동아시아 타이거즈 비공식 공연에 당신을 초대합니다."  
  > "장소: 기억의 무대 (MEMORY STAGE)"
- 🚪 **입장 버튼**: "지금 입장하기" → `/memory-stage`로 이동

### 4️⃣ 기억의 무대 (MemoryStage)
**8bit 무드의 픽셀 공연 씬**

#### 🎬 연출 시퀀스 (총 21초)
1. **조명 켜짐** (1초) - 스포트라이트 fade-in
2. **극단이 등장** (2.5초) - 🐙 문어 캐릭터 + 기타 픽 등장
3. **자막 1** (4초) - "너도 이제, 우리의 노래 안에 있구나."
4. **자막 2** (6.5초) - "Welcome to MEM✦ORY STAGE."
5. **타이거즈 실루엣** (9초) - 🎤🎸🥁🎹 멤버들 blur 효과로 등장
6. **수집한 아이템 조명** (11초) - 플레이어가 모은 아이템들이 무대를 비춤
7. **스포트라이트 강화** (15초) - 조명 밝기 증가
8. **별빛 효과** (18초) - 화면 전체 별 파티클 ✨
9. **자막 3** (21초) - "Thanks for Listening 🌙"

#### 🎁 보상 (Sequence 6 이후)
- **EXCLUSIVE BONUS** 모달 등장
- 극동아시아 타이거즈 **한정 포스터** 다운로드 기능
  - Canvas API로 동적 생성
  - PNG 형식 (800x1200px)
  - 픽셀 아트 스타일 디자인
  - 파일명: `Tigers_Secret_Poster.png`

## 🛠️ 기술 구현

### 컴포넌트 구조
```
src/
├── components/
│   ├── InviteCard.tsx          # 갤러리의 초대장 카드
│   ├── InviteScene.tsx         # 초대장 페이지 (Canvas 별 파티클)
│   └── MemoryStage.tsx         # 기억의 무대 (공연 씬)
├── utils/
│   └── posterGenerator.ts      # 포스터 동적 생성 로직
└── App.tsx                     # 라우팅 추가
```

### 주요 기술
1. **Glitch 전환 효과**
   - `filter: hue-rotate(30deg) contrast(1.3)`
   - `mix-blend-mode: difference`
   - `brightness(3)` flash

2. **Canvas Particle System**
   - `requestAnimationFrame` 기반 별 애니메이션
   - 150개의 별, 각각 독립적인 반짝임 phase
   - 파스텔 색상 배열 랜덤 적용

3. **8bit 무대 조명**
   - Framer Motion `AnimatePresence`
   - Gradient blur 효과 (`filter: blur(40px)`)
   - 다중 레이어 조명 (메인/좌/우)

4. **동적 포스터 생성**
   - Canvas API로 런타임 생성
   - 그라데이션 배경 + 별 패턴
   - 픽셀 아트 패턴 (20x20 격자)
   - Data URL → PNG 다운로드

## 🎨 디자인 컨셉

### 색상 팔레트
- **초대장**: `#fffbea` (크림) + `#ffd700` (골드)
- **밤하늘**: `#0b0c10` (딥 블랙)
- **무대**: 그라데이션 `#1a1a2e → #2d1b4e → #0b0c10`
- **조명**: 골드 `#ffd700`, 핑크 `#ff69b4`, 블루 `#87ceeb`

### 애니메이션 타이밍
- 초대장 회전: `duration: 1.5s`
- 별 반짝임: `duration: 2s, repeat: Infinity`
- 캐릭터 등장: `duration: 2s, ease: easeOut`
- 페이드 인: `duration: 2s, ease: [0.25, 1, 0.3, 1]`

## 📱 사용자 경험

### UX 플로우
1. **발견**: 갤러리에서 황금빛 카드 발견 → 호기심 유발
2. **클릭**: Glitch 효과로 놀라움 → 몰입도 증가
3. **초대**: 별이 빛나는 페이지에서 기대감 상승
4. **공연**: 수집한 아이템들이 조명으로 → 성취감
5. **보상**: 한정 포스터 다운로드 → 기념품 획득

### 감정 곡선
```
평온 (갤러리) → 흥분 (초대장 발견) → 기대 (초대 페이지) 
→ 감동 (무대 연출) → 만족 (포스터 획득)
```

## 🌈 서사적 의미

> **"너는 이제 관객이 아니라, 이 세계의 일부야."**

이 초대장은 단순한 게임 엔딩이 아닙니다.

- **관객 → 참여자**: 수동적 관람에서 능동적 경험으로
- **수집 → 연결**: 아이템을 모으는 행위가 세계와의 유대로 승화
- **팬심 → 창작**: 극동아시아타이거즈 팬으로서의 정체성을 작품 안에 녹임
- **포트폴리오 안의 공연**: 단순한 웹 페이지가 아닌, 감정을 나누는 무대
